Sobre Mim

Sou Rafael Silva, estudante de Banco de Dados em fase final de formação, com foco no desenvolvimento de APIs e sistemas back-end. Tenho experiência prática na criação de APIs utilizando Python com FastAPI, implementando autenticação com OAuth 2.0 (Password Flow), geração de tokens JWT e controle de acesso a endpoints protegidos via Bearer Token. Também aplico criptografia de senhas com bcrypt para garantir segurança no armazenamento de credenciais.

Além disso, possuo experiência com containerização utilizando Docker e integração com bancos de dados como MySQL, SQL Server e PostgreSQL, atuando na modelagem e manipulação de dados.

Tenho conhecimentos em HTML e CSS e modelagem de processos com BPMN (bpmn.io), contribuindo para uma visão mais ampla no desenvolvimento de sistemas.

Atualmente, estou expandindo minhas habilidades com Java e Spring Boot para desenvolvimento de APIs robustas e escaláveis.

Busco minha primeira oportunidade como desenvolvedor back-end júnior, onde eu possa aplicar meus conhecimentos na prática, evoluir tecnicamente e contribuir com soluções eficientes.
