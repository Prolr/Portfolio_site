# 💼 Portfólio - Rafael Silva

Meu portfólio pessoal desenvolvido com HTML e CSS  criado para apresentar minhas habilidades, projetos, experiências e formas de contato na área de desenvolvimento e banco de dados.

---

# 🚀 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar uma apresentação profissional moderna, organizada e responsiva, destacando:

- Minhas habilidades técnicas
- Projetos desenvolvidos
- Experiência acadêmica
- Tecnologias utilizadas
- Contato profissional

O site possui navegação suave entre seções e estrutura organizada para melhor experiência do usuário.
